学习笔记
1.数组
    一种线性表数据结构。
    它用一组连续的内存空间，来存储一组具有相同类型的数据。
2.链表
    数据排成像一条线一样的结构。
    每个线性表上的数据最多只有两个方向。
    除了数组，链表、队列、栈也是线性表结构。
（与线性表对立的是非线性表，比如二叉树、堆、图等。之所以叫非线性，是因为，在非线性表中，数据之间并不是简单的前后关系。）

    参考：

3.跳表
    增加了向前指针的链表叫作跳表。
    跳表是一个随机化的数据结构，实质就是一种可以进行二分查找的有序链表。
    快速查询是通过维护一个多层次的链表，且每一层链表中的元素是前一层链表元素的子集。
